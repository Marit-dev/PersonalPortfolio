Proyectos

Una colección de proyectos que muestran mis habilidades de desarrollo.

DeliverApp

Aplicación web de gestión de pedidos y entregas. Incluye autenticación, 4 roles (cliente, comercio, delivery y administrador) y paneles con distintas funcionalidades adaptados para cada tipo de usuario.

BookApp

Plataforma web para registrar, editar y gestionar libros, autores y editoriales.

ReservaCitas

Aplicación desarrollada con ASP .NET y arquitectura Onion. Cuenta con 2 roles: administrador (gestión completa de citas, horarios y estaciones) y cliente (visualización y reserva de citas en los horarios generados automáticamente).

Pokedex

Aplicación web tipo Pokédex que permite agregar, editar y eliminar Pokémon, así como gestionar su región y tipo.

Sistema Inscripción

Aplicación web simple que registra estudiantes según carrera y materia, con formulario de datos personales, selección de asignaturas y generación de un horario en tabla.
